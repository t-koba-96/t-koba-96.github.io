Creating iOS App as hobby

May 15, 2019 Project

Contents

This is just my hobby project, so it doesn’t relate to any of my research themes. I have made some iOS Applications using Xcode and Swift, so I’d like to share one of them. This app is aimed to be used as a tacticalboard, you first choose a game map and then draw the tactics using drawing mesods. After drawing , you can save them to your local or share it on SNS. You can see the demo below to know how the app works.

I dont have the license for uploading the apps to Applestore so if you want to test, download my project and build it on your local Xcode. Be carefull , the page designs are made for iPad so it might not work perfect on iPhone.

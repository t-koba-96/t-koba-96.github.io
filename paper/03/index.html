3D CNN гҒЁгҒҜ

еӢ•з”»гҒ®иЎҢеӢ•иӘҚиӯҳгҒ®гӮҝгӮ№гӮҜгҒ«гҒҠгҒ‘гӮӢжңҖиҝ‘пјҲ2018е№ҙ12жңҲзҸҫеңЁ)гҒ®гғҲгғ¬гғігғү. е…ҘеҠӣеӢ•з”»гҒ«еҜҫгҒ—гҒҰз©әй–“жғ…е ұ(2D)гҒЁжҷӮй–“жғ…е ұ(1D)гӮ’гҒҫгҒЁгӮҒгҒҰ3DгҒ®з•ігҒҝиҫјгҒҝгӮ’иЎҢгҒҶгҒ“гҒЁгҒ«гӮҲгӮҠгҖҒжҷӮз©әй–“жғ…е ұгӮ’иҖғж…®гҒ—гҒҹеӢ•з”»гҒ®иЎҢеӢ•иӘҚиӯҳгӮ’иЎҢгҒҶгҒ“гҒЁгҒҢеҸҜиғҪ(зҗҶи«–дёҠ).
еҹәжң¬зҡ„гҒ«е…ҘеҠӣгҒЁгҒ—гҒҰгҒҜгҖҒеӢ•з”»гӮ’гҒӮгӮӢжұәгҒҫгҒЈгҒҹй•·гҒ•гҒ®tгғ•гғ¬гғјгғ гҒ”гҒЁгҒ«еҲҶеүІгҒ—гҒҰе…ҘеҠӣгҒҷгӮӢ. дҫӢгҒҲгҒ°гҖҒеҫҢгҒ»гҒ©еҮәгҒҰгҒҸгӮӢC3DгғўгғҮгғ«гҒ§гҒҜеӢ•з”»е…ЁдҪ“гӮ’пј‘пј–гғ•гғ¬гғјгғ гҒ”гҒЁгҒ«еҲҶеүІгҒ—гҒҰе…ҘеҠӣгҒҷгӮӢ. еҗ„е…ҘеҠӣгҒ”гҒЁгҒ«еҮәеҠӣиЎҢеӢ•гӮҜгғ©гӮ№гӮ’1гҒӨгҒҡгҒӨеҫ—гӮӢ. еӢ•з”»е…ЁдҪ“гҒ®еҮәеҠӣиЎҢеӢ•гӮҜгғ©гӮ№гҒ®жұәе®ҡж–№жі•гҒЁгҒ—гҒҰгҒҜгҒ„гҒҸгҒӨгҒӢгҒӮгӮӢгҒҢгҖҒз°ЎеҚҳгҒӘж–№жі•гҒЁгҒ—гҒҰгҒҜеҗ„еҮәеҠӣгҒ®иЎҢеӢ•гӮҜгғ©гӮ№гҒ®е№іеқҮгӮ’гҒЁгҒЈгҒҰгӮ„гӮӢгҒ“гҒЁгҒ«гӮҲгҒЈгҒҰгҖҒжңҖгӮӮеүІеҗҲгҒ®й«ҳгҒ„иЎҢеӢ•гӮҜгғ©гӮ№гӮ’еҮәеҠӣгҒЁгҒҷгӮӢж–№жі•гҒҢгҒӮгӮӢ. д»ҘдёҠгҒ®е…ЁдҪ“гҒ®жөҒгӮҢгӮ’д»ҘдёӢгҒ«еӣігҒЁгҒ—гҒҰгҒҫгҒЁгӮҒгҒҹ.

дё»гҒӘи«–ж–Ү

3D Convolutional Neural Networks for Human Action Recognition

гғ»paper

гғ»2013е№ҙ

жңҖеҲқгҒ«3D Convolutional Neural NetworkгҒ«гӮҲгӮӢжүӢжі•гӮ’жҸҗжЎҲ.

е…ЁдҪ“еғҸ

гӮ„гҒЈгҒҰгҒ„гӮӢгҒ“гҒЁгҒЁгҒ—гҒҰгҒҜз•ігҒҝиҫјгҒҝеұӨгҒЁгӮөгғ–гӮөгғігғ—гғӘгғігӮ°еұӨгӮ’дәӨдә’гҒ®йҮҚгҒӯгҒҰгҒ„гҒҚгҖҒжҷӮзі»еҲ—гҒ®зү№еҫҙжҠҪеҮәгӮ’иЎҢгҒӘгҒҶгҒ“гҒЁгҒ§жңҖзөӮзҡ„гҒ«е…ҘеҠӣгҒ®еӢ•з”»ж•°гғ•гғ¬гғјгғ еҲҶгҒӢгӮү1ж¬Ўе…ғгҒ®зү№еҫҙгғҷгӮҜгғҲгғ«гӮ’еҫ—гӮӢ.

жң¬и«–ж–ҮгҒ§гҒҜжҷӮзі»еҲ—гҒ®з•ігҒҝиҫјгҒҝ(3D convolution)гҒ®йҡӣгҖҒйҖЈз¶ҡгҒҷгӮӢ3гғ•гғ¬гғјгғ гҒ”гҒЁгҒ«гҒҫгҒЁгӮҒгҒҰз•ігҒҝиҫјгҒҝгӮ’иЎҢгҒӘгҒЈгҒҰгҒ„гӮӢ. д»ҘдёӢгҒ®еӣігҒ®еҗҢгҒҳиүІгҒҜеҗҢгҒҳйҮҚгҒҝгӮ’иЎЁгҒ—гҒҰгҒ„гӮӢ. 2Dз•ігҒҝиҫјгҒҝгҒ®йҡӣгҒ®гӮ«гғјгғҚгғ«гӮ’гғҒгғЈгғҚгғ«ж–№еҗ‘1вҶ’3гҒ«жӢЎејөгҒ—гҒҹгҒЁиҖғгҒҲгӮҢгҒ°иүҜгҒ„.

Learning Spatiotemporal Features with 3D Convolutional Networks (C3D)

гғ»paper

гғ»2015е№ҙICCV

C3DгҒЁгҒ„гҒҶ3DCNNгҒ®гғўгғҮгғ«гӮ’жҸҗжЎҲгҒ—гҒҰгҒ„гӮӢ. 2D CNNгҒЁпј“D CNNгҒ®жҜ”ијғгҒ®еӣігҒҢгӮҸгҒӢгӮҠгӮ„гҒҷгҒ„. 2DгҒ§з•ігҒҝиҫјгӮ“гҒ жҷӮгҒЁйҒ•гҒЈгҒҰгҖҒпј“DгҒ§з•ігҒҝиҫјгӮ“гҒ еҮәеҠӣгҒҜгҒҫгҒҹ3DгҒЁгҒ—гҒҰеҮәгҒҰгҒҸгӮӢ.

C3DгҒ®е…ЁдҪ“еғҸгҒҜд»ҘдёӢгҒ®гӮҲгҒҶгҒ«гҒӘгҒЈгҒҰгҒ„гӮӢ. е…ҘеҠӣгҒЁгҒ—гҒҰгҒҜ3Г—112Г—112Г—16гғ•гғ¬гғјгғ гҖҒз•ігҒҝиҫјгҒҝгҒ«з”ЁгҒ„гӮӢгӮ«гғјгғҚгғ«гӮөгӮӨгӮәгҒҜ3Г—3Г—3гҒЁгҒӘгҒЈгҒҰгҒ„гӮӢ. гӮ«гғјгғҚгғ«гӮөгӮӨгӮәгҒҜе®ҹйЁ“зҡ„гҒӘзөҗжһңгҒ®иүҜгҒ•гӮ’еҹәгҒ«жұәе®ҡгҒ—гҒҰгҒ„гӮӢ.
3DCNNгҒҜгғ‘гғ©гғЎгғјгӮҝж•°гҒҢеӨҡгҒ„гҒҹгӮҒпјҢз•ігҒҝиҫјгҒҝеұӨгҒҢ8еұӨгҒЁгҒ„гҒҶжө…гҒ„ж§ӢйҖ пјҺ

е®ҹйЁ“

UCF101гғҮгғјгӮҝгӮ»гғғгғҲгӮ’з”ЁгҒ„гҒҰд»–жүӢжі•гҒЁзІҫеәҰгҒ®жҜ”ијғ

пј“DCNNгӮ’еҚҳдҪ“гҒЁгҒ—гҒҰз”ЁгҒ„гҒҹе ҙеҗҲгҒ®зІҫеәҰгҒҜ85.2%. иЎЁгӮ’иҰӢгӮҢгҒ°еҲҶгҒӢгӮӢгӮҲгҒҶгҒ«жңҖгӮӮе„ӘгӮҢгҒҰгҒ„гӮӢгӮҸгҒ‘гҒ§гҒҜгҒӘгҒ„.

иӘІйЎҢ

гғ»пј“DгҒ§з•ігҒҝиҫјгӮҖгҒҹгӮҒгҖҒеӯҰзҝ’гҒҷгӮӢеҝ…иҰҒгҒҢгҒӮгӮӢгғ‘гғ©гғЎгғјгӮҝж•°гҒҢеӨҡгҒ„гҖҖгҖҖ гҖҖвҶ’ж·ұгҒ„ж§ӢйҖ гҒ«еҮәжқҘгҒӘгҒ„(8еұӨпјү гҖҖвҶ’гӮҲгӮҠеӨ§иҰҸжЁЎгҒӘгғҮгғјгӮҝгӮ»гғғгғҲгҒҢеҝ…иҰҒ

Quo Vadis, Action Recognition? A New Model and the Kinetics Dataset (I3D)

гғ»paper

гғ»2017е№ҙCVPR

гғ»и©ігҒ—гҒҸгҒҜгҒ“гҒ“

дёҠиЁҳгҒ®иӘІйЎҢгӮ’и§ЈжұәгҒҷгӮӢгҒҹгӮҒгҖҒ KineticsгғҮгғјгӮҝгӮ»гғғгғҲе‘јгҒ°гӮҢгӮӢгҖҒиЎҢеӢ•иӘҚиӯҳз”ЁгҒ®ж–°гҒ—гҒ„еӨ§иҰҸжЁЎгҒӘеӢ•з”»гғҮгғјгӮҝгӮ»гғғгғҲгӮ’з”ЁгҒ„гҖҒ3DCNNгҒ®зІҫеәҰеҗ‘дёҠгӮ’е®ҹзҸҫ. ж–°гҒ—гҒ„пј“DCNNгҒ®гғўгғҮгғ«гҒЁгҒ—гҒҰ I3DгӮ’жҸҗжЎҲ.
I3DгҒ®е…ҘеҠӣгҒҜ3Г—224Г—224Г—64гғ•гғ¬гғјгғ пјҺC3D(3Г—112Г—112Г—16гғ•гғ¬гғјгғ )гҒ«жҜ”гҒ№гҒҰе…ҘеҠӣз”»еғҸгҒҜй«ҳи§ЈеғҸеәҰпјҢ1еәҰгҒ«е…ҘеҠӣгҒ§гҒҚгӮӢжҷӮй–“й•·гҒҜ4еҖҚгҒ«гҒӘгҒЈгҒҰгҒ„гӮӢпјҺ

зү№еҫҙгҖҖгҖҖ

гғ»еӢ•з”»гҒ®еӨ§иҰҸжЁЎгғҮгғјгӮҝгӮ»гғғгғҲKineticsгҒ®жҸҗжЎҲпјҲеӢ•з”»зүҲImageNet)

гғ»2DCNNгӮ’3DCNNгҒ«жӢЎејөгҒҷгӮӢгҒЁгҒ„гҒҶзҷәжғігҒ«гӮҲгӮҠпјҢC3DгҒ«жҜ”гҒ№гҒҰеӨ§е№…гҒӘгғ‘гғ©гғЎгғјгӮҝгҒ®еүҠжёӣпјҲгғ‘гғ©гғЎгғјгӮҝж•°гҖҖC3D:79M , I3D:25M пјү

гғ»дёҠиЁҳгҒ®пј’гҒӨгҒ®иҰҒеӣ гҒ«гӮҲгӮҠпјҢж·ұгҒ„ж§ӢйҖ гҒҢеҸҜиғҪгҒ«пјҲC3D:8еұӨ , I3D:22еұӨпјү

гғ» ImageNetзӯүгҒ®2DCNNз”ЁгҒ®гғҚгғғгғҲгғҜгғјгӮҜгҒ«гӮҲгӮӢPretrainгҒҢеҸҜиғҪгҒ«(еҗҢгҒҳз”»еғҸгӮ’иӨҮж•°йҮҚгҒӯгҒҰеӢ•з”»гҒЁиҰӢз«ӢгҒҰгӮӢ)

гғ»з”»еғҸгҒЁгғ•гғӯгғјгҒ®Two-StreamгӮ’з”ЁгҒ„гӮӢгҒ“гҒЁгҒ§зІҫеәҰгҒ®еҗ‘дёҠ(д»ҘдёӢгҒ®е®ҹйЁ“зөҗжһңеҸӮз…§)

е®ҹйЁ“

UCF-101гҖҒHMDBвҲ’51гҖҒKineticsгҒ®пј“гҒӨгҒ®гғҮгғјгӮҝгӮ»гғғгғҲгӮ’з”ЁгҒ„гҒҰзІҫеәҰгҒ®жҜ”ијғ. (e)Two-Stream I3DгҒҢжң¬и«–ж–ҮгҒ®жҸҗжЎҲжүӢжі•. е…ҘеҠӣгҒЁгҒ—гҒҰз”»еғҸгҒ®гҒҝгҖҒгӮӘгғ—гғҶгӮЈгӮ«гғ«гғ•гғӯгғјгҖҒдёЎж–№(Two-Stream)гӮ’з”ЁгҒ„гҒҹе ҙеҗҲгҒ«гҒӨгҒ„гҒҰгӮӮжҜ”ијғгҒ—гҒҰгҒ„гӮӢ.

е…ҘеҠӣгҒҢRGBгҒ®гҒҝгҒ®е ҙеҗҲгҒ§жҜ”ијғгҒ—гҒҹжҷӮstate-of-the-art. гҒқгҒ“гҒ«гӮӘгғ—гғҶгӮЈгӮ«гғ«гғ•гғӯгғјжғ…е ұгӮ’еҠ гҒҲгӮӢгҒ“гҒЁгҒ§гҖҒTwo-Stream 2D Cnn гҒ®жҷӮгҒ®гӮҲгҒҶгҒ«зІҫеәҰеҗ‘дёҠгҒҢеҸҜиғҪ.

3D CNN ж„ҹжғігғ»гӮігғЎгғігғҲгҖҖгҖҖ

гғ»зҸҫеңЁгҒ®state-of-the-artпјҲжңүеҗҚгҒӘгғҮгғјгӮҝгӮ»гғғгғҲгҒ«гҒҠгҒ„гҒҰгҒҜпјү

гғ»еӨ§иҰҸжЁЎгғҮгғјгӮҝгӮ»гғғгғҲKinetics(з”»еғҸгҒ§гҒ„гҒҶImagenet)гҒ®зҷ»е ҙгҒ«гӮҲгӮҠзІҫеәҰеҗ‘дёҠ

гғ» Inception-1vгӮ„ResNetгӮ’3DгҒ«жӢЎејөгҒҷгӮӢ3DCNNгғўгғҮгғ«гҒ®зҷ»е ҙгҒ«гӮҲгҒЈгҒҰпјҢImageNetгҒ§гӮӮPretrainгҒҢеҸҜиғҪгҒ«

гғ»еӯҰзҝ’гҒ«гҒӢгҒӢгӮӢжҷӮй–“гҒҜй•·гҒқгҒҶ

гғ»3D convolutionгҒ§иҖғж…®гҒ§гҒҚгӮӢжҷӮй–“жғ…е ұгҒҜгҒқгӮ“гҒӘгҒ«й•·гҒҸгҒӘгҒ„(C3DгҒ§16гғ•гғ¬гғјгғ пјҢI3DгҒ§64гғ•гғ¬гғјгғ гҒЁгҒӢ)

гғ»3D convolutionгҒ§жҷӮй–“жғ…е ұгӮӮз•ігҒҝиҫјгӮ“гҒ§гҒ„гӮӢгҒ®гҒ«й–ўгӮҸгӮүгҒҡгҖҒжӣҙгҒ«гӮӘгғ—гғҶгӮЈгӮ«гғ«гғ•гғӯгғјгҒЁгҒ®two-streamгҒ«гҒҷгӮӢгҒ“гҒЁгҒ§зІҫеәҰеҗ‘дёҠ гҖҖвҶ’зөҗеұҖжҷӮй–“жғ…е ұгҒҜгҒ©гҒ“гҒ§иҰӢгҒҰгҒ„гӮӢгҒ®пјҹ

гғ»3D convolutionгҒҜзөҗеұҖжҷӮй–“жғ…е ұгӮ’иҰӢгҒҰгҒ„гӮӢгҒ®гҒ§гҒҜгҒӘгҒҸгҒҰгҖҒе…ҘеҠӣгҒ®ж•°гғ•гғ¬гғјгғ гҒ®дёӯгҒӢгӮүжңҖгӮӮиӯҳеҲҘгҒ—гӮ„гҒҷгҒ„йғЁеҲҶ(гғ•гғ¬гғјгғ )гҒ®зү№еҫҙгӮ’жҠҪеҮәгҒ—гҒҰгҒ„гӮӢгҒ гҒ‘гҒ§гҒҜгҖҒгҒҝгҒҹгҒ„гҒӘж„ҸиҰӢгӮӮиҰӢгҒҹ.

гғ»гғ•гғ¬гғјгғ еҚҳдҪҚгҒ®иӘҚиӯҳгҒ«з”ЁгҒ„гҒҰгӮӮеҗҢж§ҳгҒ®зІҫеәҰгӮ’еҮәгҒӣгӮӢгҒ®гҒӢжӨңиЁјгҒ—гҒҹгҒ„пјҺ
